O que é Git?

Git é uma ferramenta de controle de versão que salva o histórico
de todas as alterações feitas no código. Com ele, nunca perdemos
o que foi feito e conseguimos voltar para qualquer versão anterior
do projeto a qualquer momento.

O que é GitHub?

GitHub é um serviço online onde armazenamos os repositórios Git.
Enquanto o Git funciona localmente na nossa máquina, o GitHub
guarda o código na nuvem e permite colaborar com outras pessoas.

O que é um commit?

Um commit é como uma foto do projeto em um momento específico.
Cada vez que fazemos um commit, estamos dizendo ao Git:
"salva o estado atual do código com essa mensagem explicando
o que foi feito". Isso cria um histórico de mudanças que podemos
consultar ou desfazer quando precisar.

Por que usamos Git?

- Para nunca perder código
- Para trabalhar em equipe sem sobrescrever o trabalho do outro
- Para registrar o que foi alterado, quando e por quê
- Para criar branches e testar coisas novas sem quebrar o projeto principal
